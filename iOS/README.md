# Balakun iOS SDK (Sources)

Swift 6 SDK source layout for integrating Balakun chat agents in iOS and macOS apps.

## Folder Layout

- `Sources/BalakunMobileSDK` - SDK library implementation
- `Tests/BalakunMobileSDKTests` - SDK test suite

## Package Entry Point

Use repository-root `Package.swift` (`/Volumes/Work/Projects/Balakun-SDK/Package.swift`).

## Development

```bash
cd /Volumes/Work/Projects/Balakun-SDK
swift test
./scripts/lint.sh
```
